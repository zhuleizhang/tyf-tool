## 🔧 常见问题排查

### 应用无法启动
1. 检查Node.js版本（建议16+）
2. 运行 `npm install` 重新安装依赖
3. 运行 `npm run build` 重新构建
4. 检查控制台错误信息

### OCR识别失败
1. 检查图片格式是否支持
2. 检查图片文件是否损坏
3. 检查网络连接（如使用在线OCR）
4. 尝试重置OCR引擎

### Excel导出失败
1. 检查磁盘空间是否充足
2. 检查文件保存权限
3. 确保Excel文件未被其他程序占用
4. 检查图片数据完整性

### 性能问题
1. 减少同时处理的图片数量
2. 关闭其他占用内存的应用
3. 检查图片文件大小
4. 使用性能监控面板观察资源使用

## 🎯 性能基准

**推荐测试配置：**
- 图片数量：10-50张
- 单张图片大小：< 5MB
- 图片分辨率：1920x1080以下
- 识别语言：中文+英文

**性能指标：**
- 单张图片识别时间：< 10秒
- 批量处理不应导致界面卡顿
- 内存使用：< 1GB
- CPU使用：< 80%
